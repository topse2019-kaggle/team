## 【有効性評価の結果報告】
### TopSE 2019年度 SW開発実践演習
#### 「深層学習を用いた画像分類モデルの構築パターンの抽出」におけるパターン適用の有効性評価
=======================================================

### はじめに
- このフォルダに格納されているファイルは、TopSE 2019年度 SW開発実践演習の演習テーマ「深層学習を用いた画像分類モデルの構築パターンの抽出」の演習成果物のうち、「抽出パターン適用ガイドライン」の有効性評価を行った際に作成したものである。

### 目的
- 演習に取り組んだメンバーが、本活動により抽出したデータセットとアーキテクチャによるパターン適用の効果を検証する為の判断材料とする。
